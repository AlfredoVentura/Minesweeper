# Minesweeper

Implementation of the classic Minesweeper game developed in Java using the Swing graphical interface library.

The game recreates the traditional puzzle where the player must reveal all safe cells while avoiding hidden mines. Numbers displayed on the board indicate how many mines are located in the surrounding cells.

🎮 About the Game

Minesweeper is a logic-based puzzle game where the objective is to uncover all cells that do not contain mines.

Game rules:

Each cell may contain a mine or be empty.

When a cell is revealed, it shows a number representing how many mines are adjacent to it.

Using these numbers, the player must deduce where the mines are located.

Clicking on a mine ends the game.

🎯 Gameplay

Left Click → Reveal a cell

Right Click → Mark a suspected mine (flag)

Numbers indicate how many mines are adjacent to that cell

Win Condition

Reveal all cells that do not contain mines.

Lose Condition

Click on a mine.
